C#OS is an Arch Linux distribution that positions itself as a friendly operating system for those who are intimidated by scary command lines. 
Are you a beginner or unsure about which arguments a command requires? The help command is always there to lend a hand! Don't like the shell and want to jump straight into vanilla Arch? No problem! 
The OS is built using ArchISO.

This is my first serious project on GitHub, and I'm really happy that I managed to bring it to life!
